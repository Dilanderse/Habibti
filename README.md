# Invitation 20 ans de mariage

Site d'invitation React/Vite pour un anniversaire de mariage surprise.

## Prérequis

- Node.js 20+ recommandé
- npm

## Démarrage local

1. Installer les dépendances:
   `npm install`
2. Lancer en développement:
   `npm run dev`
3. Ouvrir:
   `http://localhost:3000`

## Scripts utiles

- `npm run dev` démarre le serveur local
- `npm run build` génère la version production dans `dist/`
- `npm run preview` prévisualise le build
- `npm run lint` vérifie TypeScript

## Personnalisation rapide

- Contenu principal: `src/App.tsx`
- Métadonnées HTML (SEO + partage): `index.html`
- Styles globaux: `src/index.css`
